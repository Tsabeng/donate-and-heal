Etapes pour faire fonctionner le projet 
```sh
# Step 1: Clone 
git clone https://github.com/Tsabeng/donate-and-heal

# Step 2: Navigation vers le projet 
cd donate-and-heal

# Step 3: Installation des depandances 
npm i

# Step 4: Lancer le serveur de developpement 
npm run dev
```



## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
